# Projeto: Dados dos Atletas 🏃‍♂️

Segundo projeto de certificação do curso **LAB365 (SENAI)**. O objetivo foi evoluir a lógica anterior para uma estrutura de **Classes e Objetos** em JavaScript.

## 🎯 Objetivo

Desenvolver uma aplicação que utilize Programação Orientada a Objetos (POO) para organizar dados de atletas, calcular categorias baseadas na idade, determinar o IMC e processar médias de desempenho.

## 🛠️ Conceitos Aplicados

- **Classes e Instâncias**: Criação da classe `Atleta` com `constructor`.
- **Encapsulamento**: Uso de métodos para processar e retornar informações do objeto (`this`).
- **Tratamento de Dados**: Uso de `.slice()` para evitar a mutação de arrays originais durante a ordenação.
- **Lógica Condicional**: Estrutura `if/else if` para classificação por faixas etárias.

## 🚀 Como rodar o projeto

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório.
3. Execute o comando: `node dados-ateltas.js`
